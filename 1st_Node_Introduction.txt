What is Node js :-

1) Node is not a Language.
2) This is a Server Environment.
3) Node js can connect with database.
4) Code and Syntext is very similar to JavaScript.
5) But not exactly the same.
6) Node js is free, open-source
7) Node js use Chromes's V8 engine to execute code.




Why do we use Node :- 

1) Node is mostly used for API.
2) So we can connect the same database with Web App, Mobile App.
3) Node js is easy to understand who know JavaScript.
4) Node js is super-fast for APIs.
5) With Node and Javascript, you can becomde full-stack developer.





JavaScript & Node are the Same?

1) JavaScript and Node js code syntext is same.
2) If you know Javascript you can easily
understand Node.
3) But both are not exactly the same.
4) You can not connect Javascript to Database.
5) Node can connect with Database.
6) Node js run on the server side.
7) Javascript is run on the browser.
